# ProtoFPGA
![UFSC](https://img.shields.io/badge/UFSC-Araranguá-blue)
![Focus](https://img.shields.io/badge/Focus-FPGA%20Development-orange)

**ProtoFPGA** is the academic league at **UFSC** dedicated to rapid prototyping of complex hardware. Mentored by professors **Ney** and **Rodrigo**, our focus is to turn theory into real hardware through a collaborative learning ecosystem.

---

### Our Mission

Our goal is to work on real-world projects using FPGAs and build an open-source repository with **VHDL** and **Verilog** tutorials — high-quality material created by learners, for learners.

> [!IMPORTANT]
> **No prior experience is required to join.** We value interest and the will to learn above all else.

---

### Repositories

| Repository | Description |
|------------|-------------|
| [tutorials](https://github.com/LigaProtoFPGA/tutorials) | Step-by-step guides for VHDL, Verilog, and FPGA toolchains |

| [templates](https://github.com/LigaProtoFPGA/templates) | Standard project templates to get started quickly |
